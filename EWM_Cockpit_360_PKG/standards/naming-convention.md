# Naming Convention

## Prefixos

| Tipo | Padrão |
|---|---|
| Tabela | ZTEWM_C360_* |
| CDS Interface | ZI_EWM_C360_* |
| CDS Consumption | ZC_EWM_C360_* |
| Classe | ZCL_EWM_C360_* |
| Interface ABAP | ZIF_EWM_C360_* |
| Serviço UI | ZUI_EWM_C360_* |
| Role | Z_EWM_C360_* |
| Authorization Object | Z_EWM_C36* |
| Log Object | ZEWM_C360 |

## Regras

- Usar nomes técnicos legíveis.
- Não usar abreviação ambígua.
- Manter alinhamento entre tabela, CDS, classe e serviço.
- Toda action deve possuir `ACTION_ID` documentado.
