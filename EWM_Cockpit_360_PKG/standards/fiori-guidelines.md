# Fiori Guidelines

## Princípios

- Fiori Elements como padrão.
- Freestyle ou extensão somente com justificativa.
- Botões críticos exigem autorização backend.
- Mensagens devem ser claras para operação.
- Navegação por semantic object/action documentada.

## Áreas

- Overview
- Inbound
- Outbound
- HU
- WT
- Exceptions
- KPIs
- Reprocessing
- Action Log
- Diagnostic
